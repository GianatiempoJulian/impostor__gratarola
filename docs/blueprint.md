# **App Name**: El Impostor - Gratarola

## Core Features:

- Custom Word List Creation: Allows users to create and store a custom list of words using local storage or cookies.
- Predefined Topic Lists: Provides access to pre-built lists of topics for quicker gameplay.
- Player Configuration: Enables users to set the number of players (3-15), input player names, and select the number of imposters (1 or 2) with adjustable probability for 2 imposters.
- Role Assignment and Word Reveal: Assigns roles (impostor or word) to players and allows each player to reveal their assignment privately.
- Discussion Phase Trigger: Triggers a discussion phase screen with a text box and a button to reveal the impostor(s).
- Game End and Round Management: Displays the game results (impostor(s) and hidden word) and offers options to start a new round with the same players or restart the game.

## Style Guidelines:

- Primary color: Dark Indigo (#4B0082) to convey a sense of mystery and suspense, appropriate to the subject matter.
- Background color: Very dark gray (#222222), creating a minimal and dark atmosphere without being pure black.
- Accent color: Deep Purple (#9400D3), for buttons and important UI elements, to create contrast and focus user attention.
- Body and headline font: 'Roboto' (sans-serif) for a clear and readable user interface, as per the user request.
- Use simple, flat icons to represent different actions and game states, consistent with a minimalist design.
- Employ a clean and straightforward layout to ensure ease of use and intuitive navigation.
- Incorporate subtle animations for transitions and reveals to enhance user experience and maintain engagement.
